## Learnings
- Supabase Management API doesn't support transactions — migrations run each statement individually
- ESLint v9 dropped --ext flag entirely; flat config only
- Next.js 16 deprecated middleware.ts in favor of proxy
- Expo Router uses file-based routing like Next.js App Router
- expo-av outputs m4a (AAC) on iOS, needs audio/mp4 MIME type
- FormData in React Native needs uri/name/type object, not Blob
- Puppeteer in devcontainer needs system Chromium (/usr/bin/chromium), not bundled one (ARM/Rosetta issue)
- Figma page captures require capture script injection + hash URL parameters — Playwright MCP will simplify this
